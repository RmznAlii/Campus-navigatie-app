# Campus Navigatie App

Een simpele webapplicatie voor campus navigatie.

## Beschrijving
Deze app helpt gebruikers om zich te oriënteren op de campus en locaties te vinden.

## Structuur
- `index.html` - Hoofdpagina
- `css/style.css` - Stijlen
- `js/script.js` - JavaScript functionaliteit
- `data/locaties.json` - Campuslocaties data
- `images/` - Afbeeldingen

## Technologie
- HTML5
- CSS3
- Vanilla JavaScript
